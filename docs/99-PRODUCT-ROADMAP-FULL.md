# NAISSER 프로덕트 로드맵 — 월드클래스까지의 전체 항목

> Opus 10.0 수준의 통찰. "이건 해야 되는 거 아님?" 이 말이 나오기 전에 다 적었음.

---

## A. 피드 & 콘텐츠 (31개)

### A1. 필수 — 지금 없으면 이상한 것
- [ ] **무한스크롤** — 커서 기반 페이지네이션 + 스크롤 끝에 자동 로드
- [ ] **"새 글이 있습니다" 플로팅 배너** — X처럼 상단에 "↑ 새 글 3개" 탭하면 최상단
- [ ] **스크롤 위치 복원** — 상세 갔다 돌아오면 피드 위치 유지
- [ ] **링크 프리뷰 카드** — URL 붙여넣으면 OG이미지+제목+설명 자동 생성
- [ ] **@멘션** — `@김하늘` 입력하면 자동완성 드롭다운 + 알림 연동
- [ ] **리치 텍스트** — 굵게, 리스트, 인용(>) 마크다운 경량 지원
- [ ] **글 수정/삭제** — 수정 시 "수정됨" 표시 + 수정 이력
- [ ] **글 임시저장** — 쓰다가 뒤로가면 "임시저장할까요?" + 초안 목록
- [ ] **이미지 다중업로드 개선** — 드래그 순서 변경 + 크롭 + 압축
- [ ] **동영상 임베드** — YouTube/네이버TV URL 붙이면 인라인 플레이어

### A2. 있으면 확 달라지는 것
- [ ] **트렌딩 해시태그** — 피드 상단 "지금 뜨는 태그" 3개 표시
- [ ] **읽기 시간 추정** — 긴 글에 "약 3분 읽기" 표시
- [ ] **콘텐츠 접기** — 200자 이상이면 "더 보기" 토글 (현재 자르기만 함)
- [ ] **예약 게시** — "내일 오전 9시에 게시" 타이머 기능
- [ ] **스레드 연속 작성** — X처럼 여러 포스트를 하나의 스레드로
- [ ] **투표 결과 공유** — 투표 끝나면 결과 이미지 생성 + 공유
- [ ] **글 북마크 폴더** — 저장한 글을 "단가모음", "노하우" 등 폴더 분류
- [ ] **인용 리포스트** — 다른 글을 인용하며 코멘트 추가
- [ ] **민감 콘텐츠 가림** — 관리자가 설정한 키워드 포함 시 블러 처리

## B. 프로필 & 아이덴티티 (22개)

### B1. 필수
- [ ] **프로필 커버 사진** — 상단 배너 이미지 (학교 수업 사진 등)
- [ ] **자기소개 바이오** — 150자 짧은 소개 + 리치 텍스트
- [ ] **SNS 링크** — 링크트리 스타일. 인스타/유튜브/블로그/카카오채널 등
- [ ] **포트폴리오 갤러리** — 프로필 내 수업 사진/영상 캐러셀
- [ ] **활동 통계 공개** — "게시글 12 · 도움됐어요 89 · 팔로워 34"
- [ ] **프로필 공유 카드** — 공유하면 OG이미지 자동 생성 (이름+분야+평점)
- [ ] **QR코드 프로필** — 오프라인 명함 대신 QR로 프로필 공유
- [ ] **활동 타임라인** — 프로필에서 최근 게시글/댓글 타임라인
- [ ] **프로필 꾸미기** — 대표 색상 선택, 소개 카드 레이아웃 변경

### B2. 있으면 확 달라지는 것
- [ ] **인증 뱃지 다단계** — 브론즈/실버/골드/다이아 (서류+경력+리뷰 기반)
- [ ] **전문 분야 인증** — 교육청 자격증 등록 시 공식 인증 마크
- [ ] **강의 가능 지역 지도** — 프로필에 지도로 활동 범위 표시
- [ ] **수업 후기 하이라이트** — 베스트 리뷰 3개 프로필 상단 고정
- [ ] **올해의 활동 리포트** — 연말 "올 한 해 N개 학교에서 M시간 수업" 카드

## C. 알림 & 실시간 (15개)

### C1. 필수
- [ ] **푸시 알림** — FCM 연동 (좋아요, 댓글, 멘션, 의뢰)
- [ ] **알림 그루핑** — "김하늘 외 3명이 좋아요를 눌렀습니다"
- [ ] **딥링크** — 알림 탭하면 해당 포스트/댓글로 직행
- [ ] **알림 읽음/안읽음** — 안 읽은 알림 파란 점 + 카운트 배지
- [ ] **이메일 다이제스트** — 주 1회 "이번 주 인기 글" 요약 메일

### C2. 있으면 확 달라지는 것
- [ ] **실시간 업데이트** — WebSocket으로 새 댓글/좋아요 즉시 반영
- [ ] **타이핑 인디케이터** — 댓글 작성 중 "○○님이 입력 중..."
- [ ] **방해금지 스케줄** — 수업 시간대(9-17시) 알림 무음
- [ ] **알림 사운드 커스텀** — 나이써 전용 알림 사운드
- [ ] **알림 필터** — 좋아요만/댓글만/의뢰만 선택 수신

---

## D. 검색 & 디스커버리 (12개)

### D1. 필수
- [ ] **피드 내 검색** — 커뮤니티 글 전문 검색
- [ ] **검색 자동완성** — 해시태그, 작성자명, 인기 키워드 서제스트
- [ ] **검색 필터** — 기간, 카테고리, 지역, 좋아요순/최신순
- [ ] **검색 히스토리** — 최근 검색어 5개 + 삭제 기능

### D2. 있으면 확 달라지는 것
- [ ] **추천 피드** — 내 관심 분야/지역 기반 개인화 추천
- [ ] **"이런 강사는 어떠세요"** — 프로필 하단 관련 강사 추천
- [ ] **위치 기반 근처 강사** — GPS로 현재 위치 반경 N km 강사
- [ ] **인기 상승 중** — 최근 24시간 조회수 급등 글 하이라이트

---

## E. 신뢰 & 안전 (14개)

### E1. 필수
- [ ] **신고 처리 플로우** — 신고 → 관리자 검토 → 처리 결과 알림
- [ ] **차단/뮤트** — 특정 유저 글/댓글 안 보이게
- [ ] **프라이버시 설정** — 프로필 공개 범위 (전체/강사만/비공개)
- [ ] **댓글 필터링** — 비속어/욕설 자동 필터 + *** 처리
- [ ] **스팸 방지** — 동일 내용 반복 게시 차단, 새 계정 게시 제한

### E2. 있으면 확 달라지는 것
- [ ] **쉐도우밴** — 문제 유저 글이 본인에게만 보이게 (조용히 격리)
- [ ] **컨텐츠 어필** — 삭제된 글 이의제기 플로우
- [ ] **신뢰도 점수** — 활동 기반 숨겨진 점수 (스팸 확률 계산용)
- [ ] **2FA** — 관리자/인증 강사 대상 이중 인증

## F. 퍼포먼스 & 기술 (18개)

### F1. 필수
- [ ] **이미지 최적화** — next/image로 WebP 자동 변환 + srcSet
- [ ] **이미지 블러 프리홀더** — 저해상도 blur-up 로딩
- [ ] **가상 스크롤** — 100+ 포스트에서 DOM 노드 재활용
- [ ] **API 응답 캐싱** — SWR/React Query로 스테일 데이터 즉시 표시
- [ ] **오프라인 지원** — Service Worker + 최근 피드 캐싱
- [ ] **번들 최적화** — dynamic import로 커뮤니티 코드 분할
- [ ] **에러 트래킹** — Sentry 연동 (프론트+백엔드)
- [ ] **Core Web Vitals** — LCP < 2.5s, FID < 100ms, CLS < 0.1

### F2. 있으면 확 달라지는 것
- [ ] **A/B 테스팅** — 피처 플래그로 UI 실험
- [ ] **서버 사이드 렌더링** — SEO + 첫 로딩 속도
- [ ] **WebSocket 연결** — 실시간 좋아요/댓글 업데이트
- [ ] **인덱스 최적화** — PostgreSQL 쿼리 분석 + 인덱스 튜닝
- [ ] **CDN** — Vercel Edge + 이미지 CDN
- [ ] **로드 테스팅** — 동시 접속 1000명 시뮬레이션

## G. 접근성 & 국제화 (8개)

- [ ] **다크 모드** — 커뮤니티 전체 다크 테마 지원
- [ ] **폰트 크기 조절** — 설정에서 본문 크기 조정 (14/16/18px)
- [ ] **스크린 리더** — 모든 인터랙티브 요소에 ARIA 라벨
- [ ] **모션 감소** — `prefers-reduced-motion` 존중
- [ ] **고대비 모드** — 저시력 유저용 고대비 컬러
- [ ] **키보드 네비게이션** — Tab 순서 + Enter 활성화
- [ ] **영어 UI** — 다국어 지원 (영어 기본)
- [ ] **RTL 지원** — (미래) 아랍어 등 RTL 언어

## H. 어드민 & 운영 (16개)

### H1. 필수
- [ ] **콘텐츠 관리** — 게시글/댓글 검색, 숨기기, 삭제, 고정
- [ ] **유저 관리** — 유저 검색, 정지, 등급 변경
- [ ] **신고 대시보드** — 미처리 신고 목록 + 원클릭 처리
- [ ] **공지사항 작성** — 카테고리별 공지 + 전체 공지
- [ ] **커뮤니티 통계** — DAU/MAU, 게시글수, 댓글수, 좋아요수 차트

### H2. 있으면 확 달라지는 것
- [ ] **자동 모더레이션** — AI 기반 유해 콘텐츠 감지
- [ ] **AB 테스트 관리** — 어드민에서 실험 생성/종료
- [ ] **광고 관리** — 피드 내 광고 등록/예약/통계
- [ ] **키워드 모니터링** — 특정 키워드 포함 글 자동 알림
- [ ] **커뮤니티 가이드라인** — 작성 규칙 페이지 + 동의 플로우
- [ ] **유저 등급 시스템** — 활동 기반 자동 등급 (씨앗/새싹/나무/숲)

## I. 감성 디자인 & 리텐션 (11개)

- [ ] **마일스톤 축하** — 첫 글 작성, 첫 좋아요 100, 첫 인증 시 축하 애니메이션
- [ ] **연속 활동 스트릭** — "5일 연속 활동 중 🔥" 프로필 뱃지
- [ ] **도움왕 뱃지** — 월간 도움됐어요 TOP3 특별 뱃지
- [ ] **올해의 활동 리뷰** — 연말 "2026년 나의 활동" 요약 카드
- [ ] **온보딩 첫 글 유도** — 가입 후 "첫 글을 작성해보세요" 가이드
- [ ] **빈 피드 → 추천 콘텐츠** — 글이 없을 때 인기글/추천글 제안
- [ ] **이스터에그** — 특정 해시태그에 숨겨진 이펙트
- [ ] **시즌 테마** — 크리스마스, 새학기 등 시즌별 UI 테마
- [ ] **감사 알림** — "이번 주 3명이 도움됐다고 했어요" 주간 리포트
- [ ] **재방문 유도** — "어제 인기글 못 보셨죠?" 푸시
- [ ] **달성 시스템** — "첫 투표 생성", "댓글 50개 달성" 등 업적

## J. 모네타이제이션 (7개)

- [ ] **프리미엄 프로필** — 월 구독으로 프로필 상위 노출
- [ ] **프로모트 포스트** — 자기 글 유료 부스트 (노출 증가)
- [ ] **피드 네이티브 광고** — 포스트 사이 자연스러운 스폰서 카드
- [ ] **배너 광고 관리** — 어드민에서 배너 등록/타겟팅/통계
- [ ] **광고 빈도 제한** — 3포스트당 1광고 이하, 유저 닫기 가능
- [ ] **제휴 마켓플레이스** — 교구/재료 공동구매 게시판
- [ ] **수익 대시보드** — 월간 광고 수익 + 프리미엄 구독자 수
